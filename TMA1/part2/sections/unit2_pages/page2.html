JavaScript Basics

JS is a client-side scripting language that has become the backbone of modern web development. Developed in the late nineties, JS has evolved over the years to serve a number of functions. Some basic knowledge of programming is assumed for this next section.

Javascript can be directly added to a webpage by using the <script></script> tag. Inside this Html tag, any type of JavaScript can be written.
