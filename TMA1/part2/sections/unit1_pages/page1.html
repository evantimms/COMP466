Basic Tags

HTML works by generating a webpage based on tags defined in the markup language. A tag contains three parts: The opening seperator, content, and closing seperator. The opening and closing seperators are written as <tag_name></tag_name>. The content is written inside the tags and can either be basic characters, or more HTML.

The Div: <div></div>

The div is the most basic tag. It is commonly used to help structure a page. It has no special preformatting.

The Paragraph: <p></p>

The paragraph tag is commonly used to display text on a webpage since it uses preformatting. Browsers will add a blank line after a p tag.

The Head, Body and Html Tags <html></html> <body></body> <head></head>

These elements are used to define the page. Browsers will only process pages that contain a properly formatted HTML tag. All content goes inside this tag.

Head tags contain metadata about the page, as well as links to both external and internal sources (such as JS scripts and CSS stylesheets).

The body tag contains the actual HTML that will display in the browser window.
