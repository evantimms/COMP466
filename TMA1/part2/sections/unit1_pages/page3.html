Forms

Forms are the primary way of entering information into a webpage. A form element is used to define the form itself, and a number of different inputs can be used to aquire information from the user.

Input Tags

Input tags are basic elements which can accept many different forms of data from a user, wether that be words, numbers, dates etc. These are often used in applications such as login forms.

Examples of types include:

  • Text: Any ASCII backed text.
  • Password: Any ASCII backed text, but input is masked.
  • Date: A Date.
  • Radio: A radio button, which the option to be checked or unchecked.
  • File: A file upload link can be used to send files from a local computer to a server.
  • Tel: A telephone number.
