JavaScript and AJAX

Oftentimes it is important to fetch data from other websites. This can be done using client side JavaScript with a programming technique called AJAX.

AJAX is short for "Asynchronous JavaScript and XML". It is used to fetch and update a web page without reloading the page itself.
